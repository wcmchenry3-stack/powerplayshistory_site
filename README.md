# powerplayshistory.com

Coming soon site for powerplayshistory.com.

## Stack

React + Vite | Tailwind CSS | i18next (13 locales)

## Commands

- `npm run dev` — dev server
- `npm run build` — production build
- `npm run test:run` — tests (CI)
- `npm run test:coverage` — tests with coverage
- `npm run lint` — lint
- `npm run i18n:check` — check for untranslated strings
- `npm run i18n:translate` — translate with OpenAI
